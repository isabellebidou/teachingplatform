# Language Feedback Tool

## Version
 **version 1.0.0**, the first stable and functional release of the project.
 **version 1.0.1**, localized to french.
 **version 1.0.2**, name Izzy Speak English.
## Description
This project provides a basic feedback service to help users improve their language skills through interactive exercises.

## Features

### 🎤 Audio Feedback
- Users can record a short spoken statement.
- The system analyzes pronunciation.
- Users receive feedback to help improve their speaking skills.

### ✍️ Spelling Feedback
- Users can write and submit a short text.
- The system checks for spelling mistakes.
- Feedback is provided to help users correct errors.

### 📘 Grammar Practice
- Users can select a grammar topic.
- The system generates an **A1-level gap-fill quiz**.
- Users receive feedback on their answers to support learning.
